# 顺序表

# 1. 概念

定义：具有相同数据类型的 n (n≥0) 个数据元素组成的有限
序列。是最简单、最常用的数据结构。

・线性表的顺序表示就是用一组地址连续的存储单
元依次存储线性表的数据元素。
・线性表的这种机内表示称做线性表的顺序存储结
构或顺序映像。
・通常，称这种存储结构的线性表为顺序表。
