# 矩阵的压缩存储

有些阶数很高的矩阵中，有许多值相同的元素或者零元素，为了节省存储空间，可以对这类矩阵进行压缩存储。
所谓压缩存储是指：为多个值相同的元只分配一个存储空间；对零元不分配空间。假若值相同的元素或者零元素在矩阵中的分布有一定规律，则我们称此类矩阵为特殊矩阵；反之，称为稀疏矩阵。
