# 5. 归并排序

基本思想
1）把 n 个记录看成 n 个长度为 l 的有序子表；
2）进行两两归并使记录关键字有序，得到［n/2］个长度为 2 的有序子表；
3）重复第（2）步，直到所有记录归并成一个长度为 n 的有序表为止。

性能分析：
空间复杂度：O (n)
时间复杂度：O (nlog2n)
