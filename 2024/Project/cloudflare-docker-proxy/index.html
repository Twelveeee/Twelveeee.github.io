# 前言

由于众所周知的原因，docker 没办法比较好的访问。本文介绍如何使用 Cloudflare Workers 搭建 Docker 代理服务。

于是尝试使用了阿里云的镜像服务，本以为需要注册效果会好点，拉完了镜像才发现镜像是踏马 21 年的镜像....

# 关联 worker 域名

在 Cloudflare 控制台首页选择你的域名。

创建 DNS 记录，输入域名前缀和 IP 地址（IP 地址可以随便填写，关键是 Proxy 需要勾选）。

保存后，添加 Worker 的路由

格式为 xxx.example.com/*

后面这个 /* 不能省略

等待几分钟后，检查是否成功。
